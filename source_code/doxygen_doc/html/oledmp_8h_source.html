43 /**************************************************
44 * SSD1322 driver for 256x16x4 OLED display
45 *
46 * Pin Function SPI connections
47 * ----+------------+----------
48 * 1 VSS GND
49 * 2 VBAT 3.3V-5V
50 * 3 NC
51 * 4 D0 SCLK Serial Clock
52 * 5 D1 MOSI Serial Data Input
53 * 6 D2 NC
54 * 7 D3 GND
55 * 8 D4 GND
56 * 9 D5 GND
57 * 10 D6 GND
58 * 11 D7 GND
59 * 12 #RD GND
60 * 13 #WR GND
61 * 14 DC Data / Command
62 * 15 #RESET
63 * 16 #CS Chip select
64 *
65 * Note: All logic pins are 3.3V max.
66 *
67 * BS1 BS0 Mode
68 * --------+---------------------
69 * 0 0 "4 Line SPI" 8-bit + DC pin
70 * 0 1 "3 Line SPI" 9-bit DC is 9th bit
71 * 1 0 8-bit 8080 parallel
72 * 1 1 8-bit 6800 parallel
73 *
74 * Note: SPI mode is write only (MOSI)
75 *
76 **************************************************/
