Mooltipass Random Number Generation description

1- RNG-LIBRARY, USING THE LIBRARY

How to use the library? how to work with it ?

We only have to remember about two functions.

1 void rngInit(void); // Init Timer0 and WDT Interrupt to generate random numbers
2 void fillArrayWithRandomBytes(uint8_t* buffer, uint8_t nb_bytes); //fill a buffer with random bytes, it's a blocking function !

2- TESTING THE LIBRARY AND IMPLEMENTATION

We have tested the library performance generating 1 milion of bytes. The bytes were sent using mooltipass HID protocol and tools/hiddebug/hiddebug.c program and saved into a binary file.

Code extract from MP side:

1 case CMD_GET_RANDOM_NUMBER:
2  fillArrayWithRandomBytes(array, 4);
3  hexachar_to_string(array[3], &str[0]);
4  hexachar_to_string(array[2], &str[2]);
5  hexachar_to_string(array[1], &str[4]);
6  hexachar_to_string(array[0], &str[6]);
7 
8  usbPutstr((char*)str);
9  break;

To get the statistics of the binary file we have used the ENT Utility . The Fourmilab ENT program is a public domain utility which tests binary data sequences, either as a series of 8 bit bytes, or as a bit stream, with five standard tests for randomness. The output of the ENT utility with the 1 milion binary file is:
